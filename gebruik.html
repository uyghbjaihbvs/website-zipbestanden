Waar gebruik je zip bestanden voor/wie gebruikt welk zip bestand?

Zip bestanden gebruik je meestal bij het verkleinen van een bestand bijvoorbeeld bij: Muziek-bestanden en foto's. Bij sommige zip bestanden is het zo dat je een wachtwoord kan instellen zodat er niet vreemdelingen in je bestand kunnen komen. Zip bestanden kunnen op alle verschillende besturingssystemen zoals: Windows visa, Windows XP, Linux en Mac. WinRAR staat erom bekent dat het gemakkelijk in kan pakken en kan uitpakken en het is helemaal gratis dus je hoeft er niks voor te betalen. Maar je kan ook een licensie kopen voor WinRAR dat ervoor zorgt dat er grotere bestanden in je zip bestand kunnen.

Welke zip programma's zijn er allemaal?

7Zip:

7Zip is een programma waarmee je kan archiveren en comprimeren 7Zip wordt in de taal C++ geschreven.

arc:

is een programma om dingen te archiveren het was de voorloper van zip bestanden en werd gebruikt in MS-DOS en Macintosh computers

Ajr:

Ajr is een compressieprogramma die in de jaren 90 heel erg veel werd gebruikt het was voor windows 16 bit en 32 bit.

Bzip2:

Bzip2 is een compressieprogramma het is bedoeld als de opvolger van gzip dat op het besturingssysteem van Linux stond.

Gzip:

Gzip is de afkorting van GNU zip is een compressieprogramma van Linux dat bestanden comprimeert.

PKZIP:

Is een archiverings- en compressieprogramma. Het is ontwikkeld door Phil Katz. Het was mogelijk om het compressiebestand met een wachtwoord te beveiligen en als het zip bestand beschadigd is dat je een waarschuwing krijgt voor het uitpakken.

SQLite:

SQLite is meer een databaseprogramma waar je je data in kan opslaan. De aangeraden formaten zijn JSON en CSV.

winzip:

Winzip is een van de meest gebruikte zip programma het maakt gebruik van zipbestandindeling maar kan ook andere compressie-algoritmes lezen. De volgende bestandformaten worden ondersteund: ARC, ARJ, B64, BHX, BZ, BZ2, CAB, GZ, HQX, LZH, MIM, RAR, TAR, TAZ, TBZ, TBZ2, TGZ, TZ, UU, UUE, XXE, Z en ZIP.

winrar:

WinRAR is ook een van de meest gebruikte zip programma's. WinRAR maakt via algoritmen gecomprimeerde archieven. WinRAR ondersteunt de volgende bestandformaten: RAR, ZIP, CAB, ARJ, LZH, 7Zip, TAR, Gzip, UUE, BZ2, JAR, ISO en Z.

winACE:

WinACE is een heel erg verouderde archiveringsprogramma voor Windows. Het werkt uit de ACE-archiefformaat en het ondersteunt de volgende bestandformaten: ZIP, RAR en CAB. Het programma koste wel geld maar de ontwikkelaar maakte ook andere programma's zoals een freeware genaamd Unace voor MacOS en Linux.
